Garrett Masters Work

UNDP: SDG's Today

A compelling mobile first website that conveys the state of United Nations Development Programme (UNDP’s) Sustainable Development Goals (SDG’s) as it relates to the Covid-19 epidemic.

A project of Fifty and Fifty creative agency

Brief

The SDGs Today is a semi-annual web series that focuses on the question, ‘what is the state of the Sustainable Development Goals (SDGs) today’? This website focuses on SDG’s goal number three on Good Health and Well-being.

Goals & Objectives

The primary goal for this site is to make the statistical data visually engaging and accessible in english, spanish, and french languages. The micro site was to be hosted on github pages so it needed to be built as a static website.


Involvement & Responsibilities

Frontend Development

Approach & Direction

In partnering with design we kicked a few ideas back and forth until we landed on making the experience of the site feel organic. We wanted to play off of the organic shape of a microscopic cell.

When you look at a cell there is always some movement on a microscopic level. We decided to take that idea and make text and image shapes that felt like you were looking under a microscope.

Design Review

We landed on making a chapter approach that has six pages or chapters to tell this whole story. These chapters would be broken down into the following chapters:

  • A new future for health
  • Health as a barometer
  • Health and equality
  • Health and biodiversity
  • Development on fast forward
  • A global stress test

Execution

With the mobile first approach we wanted the website to be slim and not have to rely on heavy frameworks.

I built the site using webpack which translated my SCSS into CSS as well as minified the javascript for this website. This allowed us to streamline the code for production. .

Challenges

The challenge was the images that were organic shapes needed to have an overlay to display the image credit. The images had a drop shadow ‘s that needed to be dependent on the element that it was on.

Solution

The organic image shapes proved to be challenging for all the things that it was doing. After trial and error I discovered that you can manipulate boxes to have an organic shape by using a border-radius like the following: (border-radius: 71% 42% 45% 69%/25% 43% 69% 89%;)

This is the effect that we were going for to create those organic shapes. The benefit of it being border-radius is that I can tie that to an animation and make those shapes infinitely move in a slow amoeba like way.

The shapes we could make on a div which had the img element inside. Taking advantage of pseudo elements we could create an overlay and the text to pop over the image when the right action takes place.

Conclusion

I am glad that I was able to be a part of this project. Everytime I am able to work closely with design I learn something new. This definitely pushed what I thought could be possible with CSS. I am always anxious to learn more and push the boundaries.
